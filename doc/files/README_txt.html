rewrite

  • rewrite.rubyforge.org

DESCRIPTION:

  • Adds certain features to Ruby programs using sexp rewriting rather than open classes.

FEATURES:

  • Andand
  • Call by name
  • Unhygienic macros

REQUIREMENTS:

  • Parse_Tree
  • Ruby2Ruby

INSTALL:

  • sudo gem install rewrite
